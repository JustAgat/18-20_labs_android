# Task Planner (app_book_task)

Персональный планировщик задач для Android, сочетающий в себе классический UI на ViewBinding и современные возможности Jetpack Compose. Приложение поддерживает полноценную авторизацию, локальное хранение данных и детальную статистику.

## 🌟 Основные возможности

*   **Система аккаунтов**: Полноценная регистрация и авторизация пользователей через локальную БД SQLite.
*   **Управление задачами**: Создание, удаление и возможность отмечать задачи как выполненные (с визуальным зачеркиванием текста).
*   **Детальная статистика**:
    *   Общий счетчик задач (всего/выполнено/в процессе).
    *   Разбивка продуктивности по датам.
*   **Многоязычность**: Полная поддержка **Русского** и **Английского** языков (переключается вместе с системным языком).
*   **Персонализация**:
    *   Поддержка темной и светлой тем.
    *   Настройка размера шрифта.
    *   Сортировка задач по дате создания.
*   **Безопасность**: Задачи каждого пользователя хранятся изолированно и доступны только после входа.

## 🛠 Технологический стек

*   **Язык**: Kotlin
*   **Архитектура**: Single Activity + Navigation Component
*   **База данных**: SQLite (через `SQLiteOpenHelper`)
*   **UI**: XML ViewBinding + Jetpack Compose (Material 3)
*   **Сетевой слой**: Retrofit 2 + Gson (подготовлено для работы с API)
*   **Хранение настроек**: SharedPreferences

## 📦 Основные зависимости

Проект использует современные библиотеки Android Jetpack:
*   **UI**: `Material Components 1.12.0`, `AppCompat 1.7.0`, `RecyclerView 1.3.2`
*   **Navigation**: `Navigation Fragments/UI KTX 2.8.5`
*   **Compose**: `Compose BOM 2024.04.01`, `Activity Compose 1.9.3`, `Material3`
*   **Network**: `Retrofit 2.9.0` + `Converter Gson`
*   **Core**: `Core KTX 1.15.0`, `Lifecycle Runtime KTX 2.8.7`

## 🚀 Как запустить проект

1.  **Клонируйте репозиторий** или скачайте архив с проектом.
2.  **Откройте проект** в Android Studio (рекомендуется версия Ladybug или новее).
3.  **Синхронизируйте Gradle**: Нажмите `File -> Sync Project with Gradle Files`.
4.  **Выберите устройство**: Запустите эмулятор или подключите физическое устройство (Android 7.0 / API 24 и выше).
5.  **Запуск**: Нажмите кнопку `Run (Shift + F10)`.

## 📁 Структура проекта

*   `db/` — логика работы с SQLite (`DBHelper`).
*   `models/` — Data-классы для задач и пользователей.
*   `adapters/` — Адаптеры для списков (RecyclerView).
*   `ui/theme/` — Настройки Compose темы.
*   `navigation/` — Граф навигации приложения.
